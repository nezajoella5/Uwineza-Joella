        2. 2.Customer table

          3. Customer Table database structure This table stores information about each customer such as CustomerID, fname, lname,Email,Gender. It serves as a central repository for all customer-related data.


        3. 3.Address table

          4. Address table database structure This table Stores details or information about the address of the customers, including AddressID,customerID, city, country etc. This table helps in managing Accounts of customers.


        4. 4.Product table

          5. Product table database structure This table contains the type of product the customer purchase, including ProductID, CustomerID,price etc.


        5. 5.Orders table

          6. Orders table database structure This table Stores information about orders that the customers ordered and it includes OrderID,CustomerID,Totalamount,Ordertable .


        6. 6.Feedback table

        Feedback table database structure this table contains feedback of the customer about the product he/she buys and it includes FeedbackID,CustomerID,comments,date.

        Attributes of Customer Table
        Here are the descriptions or purposes of the attributes/columns used in Customer table:

        Customer_Id: The column which are represents id of product which is unique value called PRIMARY. It is set to auto-increment (AUTO_INCREMENT).
        AUTO_INCREMENT means that the database will automatically generate a unique ID for each new record, typically incrementing the previous value of Product table.

        Firstname: This column represents the first name of the customer example : Uwineza


        Lastname: This column represents the lastname of the customer example: Joella
